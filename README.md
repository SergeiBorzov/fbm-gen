FBM-Gen
=======
fbm-gen is a simple fractal brownian motion terrain generator. 
It is a project for course PV097 at Masaryk University

Author
------
Sergei Borzov

Requirements
------------
OS - Windows\
OpenGL 3.3+\
glew\
glfw\
glm\
cjson
